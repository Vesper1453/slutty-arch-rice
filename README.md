## 🛠️ Customization

### Colors
Edit the mode-specific files:

- `~/.config/i3/mode.conf` - Window manager colors
- `~/.config/polybar/config.ini` - Status bar colors
- `~/.config/alacritty/alacritty.yml` - Terminal colors
- `~/.config/rofi/config.rasi` - Launcher colors
- `~/.config/dunst/dunstrc` - Notification colors

### Transparency
Edit `~/.config/picom/picom.conf`:

```conf
inactive-opacity = 0.88;
active-opacity = 0.95;
```

### Gaps
Edit `~/.config/i3/mode.conf`:

```conf
gaps inner 8
gaps outer 0
```

## 🎭 Mode Switcher Script


What it does:
1. Updates i3 color configuration
2. Switches Polybar theme
3. Switches Alacritty theme
4. Switches Rofi theme
5. Updates Dunst notification colors
6. Adjusts Picom transparency levels
7. Sets mode-appropriate wallpaper
8. Reloads all components
